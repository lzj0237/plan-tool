# PlanPilot 用户指南

> 简洁高效的个人任务管理工具

---

## 快速开始

### 1. 访问应用

打开浏览器访问：**http://localhost:8000/web/index.html**

（或者直接访问 http://localhost:8000/docs 查看 API 文档）

### 2. 注册账号

如果是新用户，点击「注册」标签，填写：
- 邮箱：你的邮箱地址
- 密码：至少6位
- 名称：你的显示名称

注册成功后会自动登录。

### 3. 登录

已有账号直接输入邮箱密码登录。

## 功能说明

### 添加任务

1. 在顶部输入框输入任务名称
2. （可选）选择计划执行日期
3. 点击「添加」按钮

### 任务状态

| 状态 | 说明 | 可执行操作 |
|------|------|-----------|
| 待开始 | 任务已创建但未开始 | 开始 |
| 进行中 | 任务正在执行中 | 暂停、完成 |
| 已暂停 | 任务暂停执行 | 继续、完成 |
| 已完成 | 任务已结束 | 删除 |

### 操作任务

- **开始**：将任务状态改为「进行中」，开始计时
- **暂停**：将任务状态改为「已暂停」，暂停计时
- **继续**：从暂停状态恢复为「进行中」
- **完成**：结束任务，状态改为「已完成」
- **删除**：永久删除任务

### 统计面板

顶部卡片显示：
- **总任务**：当前所有任务数量
- **已完成**：已完成的任务数
- **进行中**：正在执行的任务数
- **待开始**：尚未开始的任务数

## 数据安全

- 每个用户有独立的租户空间，数据完全隔离
- 密码使用哈希存储，不会泄露
- 任务数据存储在本地 PostgreSQL 数据库

## 常见问题

### Q: 为什么计时不准确？
A: 计时基于你点击「开始」「暂停」「完成」的时间戳，请确保操作时网络畅通。

### Q: 如何查看历史任务？
A: 在任务列表中向下滚动，已完成的任务会保留在列表中。

### Q: 能修改已创建的任务吗？
A: 目前暂不支持修改任务内容，可删除后重新创建。
